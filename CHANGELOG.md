# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-11-24

### Added
- Initial release
- Core sanitization functions (`sanitizeInput`, `sanitizeHtml`, `detectXSS`)
- React components and hooks (`SafeHtml`, `SafeText`, `useSafeInput`, `withSanitizedProps`)
- Vue 3 composables and directives (`useSafeInput`, `v-sanitize`)
- Svelte action (`sanitize`)
- Form library adapters (Zod, Yup, React Hook Form)
- TypeScript support with full type definitions
- CommonJS and ESM builds
- Comprehensive test suite
- Documentation and examples
